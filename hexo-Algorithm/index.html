一、基本数据结构

二叉树

二叉树节点的深度：根节点到该节点的最长简单路径边的条数或者节点数。
二叉树节点的高度：该节点到叶子节点的最长简单路径边的条数或者节点数。

前序遍历求深度，后序遍历求高度。

二、常见算法

二叉树遍历

二叉树遍历方式分为:深度优先遍历、广度优先遍历

深度优先遍历：

  • 前序遍历（递归法，迭代法）；遍历顺序：中 左 右
  • 中序遍历（递归法，迭代法）；遍历顺序：左 中 右
  • 后序遍历（递归法，迭代法）；遍历顺序：左 右 中

广度优先遍历：

  • 层次遍历（迭代法）

深度优先遍历-递归法

递归法一般构建思路：

  • 确定递归函数的参数与返回值。哪些参数是需要递归过程处理的，返回值的类型是什么
  • 确认递归终止条件。
  • 确定单层递归的逻辑。每一次递归要进行的操作

此处以前序遍历为例，中序和后序遍历仅需改变处理逻辑部分的顺序。

class Solution {
public:
    void pretravel(TreeNode*cur, vector<int> & vec){
        if(cur == NULL) return;
        vec.push_back(cur->val);                   // 当前节点的数值导入
        pretravel(cur->left,vec);                  // 查找左节点
        pretravel(cur->right,vec);                 // 查找右节点
    }

    vector<int> preorderTraversal(TreeNode* root) {
        vector<int> res;
        pretravel(root, res);
        return res;
    }
};

深度优先遍历-迭代法

借助栈实现，三种遍历方法不同。
前序遍历: 先将根节点放入栈中，然后将右孩子加入栈，再加入左孩子。
因为栈是后进先处理，先放入右孩子，再放入左孩子。

class Solution {
public:
    vector<int> preorderTraversal(TreeNode* root) {
        vector<int> res;
        stack<TreeNode*> st;
        st.push(root);
        if(root == NULL) return res;
        while(!st.empty()){
            TreeNode* cur = st.top();                         // 中节点
            st.pop();
            res.push_back(cur->val);
            if(cur->right != NULL) st.push(cur->right);       // 右节点
            if(cur->left != NULL) st.push(cur->left);         // 左节点
        }
        return res;
    }
};

后序遍历: 可以按照中，左，右的顺序放入，然后反转最后的结果，代码参考前序遍历。

中序遍历：整个过程涉及两个操作，访问节点和处理节点，访问节点是遍历节点，处理节点是将元素放入res数组中。中序遍历两个过程是分开的，需要借助指针帮助访问节点，栈来帮助处理节点上的元素。

class Solution {
public:
    vector<int> inorderTraversal(TreeNode* root) {
        vector<int> res;
        stack<TreeNode*> st;
        TreeNode* cur = root;                     // 指针记录当前位置
        while(cur != NULL || !st.empty()){
            if(cur != NULL){
                st.push(cur);                     // 如果当前位置不是空指针，放入栈
                cur = cur->left;                  // 指针指向左孩子
            }else{
                cur = st.top();                   // 如果是空指针，处理栈顶元素
                st.pop();
                res.push_back(cur->val);
                cur = cur->right;
            }
        }
        return res;
    }
};

深度优先遍历-统一迭代法

问题核心：访问节点与处理节点不一致的问题。

解决思路：访问节点和处理节点都放入栈中，但要把处理的节点做标记，即在后面加入空指针。

class Solution {
public:
    vector<int> inorderTraversal(TreeNode* root) {
        vector<int> res;
        stack<TreeNode*> st;
        if(root != NULL) st.push(root);
        while(!st.empty()){
            TreeNode* cur = st.top();
            if(cur != NULL){
                st.pop();
                if(cur->right) st.push(cur->right);              // 加入右孩子

                st.push(cur);                                  // 加入待处理变量
                st.push(NULL);                                 // 加入标记
                
                if(cur->left) st.push(cur->left);              // 加入左孩子
            }else{
                st.pop();
                cur = st.top();
                res.push_back(cur->val);
                st.pop();
            }
        }
        return res;
    }
};

此处以中序遍历（右中左）为例，前序遍历（右左中）和后序遍历（中右左）仅需更改节点加入顺序。

广度优先遍历

遍历方式：就是从左到右一层一层的去遍历二叉树。

数据结构：队列，先进先出，符合一层一层遍历的逻辑。

class Solution {
public:
    vector<vector<int>> levelOrder(TreeNode* root) {
        vector<vector<int>> res;
        queue<TreeNode*> que;
        if(root != NULL) que.push(root);
        while(!que.empty()){
            int length = que.size();
            vector<int> vec;
            for(int i = 0 ;i < length;i++){
                TreeNode* cur = que.front();
                que.pop();
                vec.push_back(cur->val);                        // 处理节点
                if(cur->left) que.push(cur->left);              // 加入左节点
                if(cur->right) que.push(cur->right);            // 加入右节点
            }
            res.push_back(vec);
        }
        return res;
    }
};

注意点：返回值数据结构为vector<vector<int>>

三、C++项目构建

C++编译环境安装

CMake使用方法

1、创建CMakeLists.txt文件
在项目根目录文件夹下创建CMakeLists.txt的文件，并包含以下内容：

cmake_minimum_required(VERSION 3.10)  // 指定CMake的最小版本要求

project(MyProject)                   // 设置项目名称

add_executable(MyExecutable main.cpp)   // 添加可执行文件并指定源文件

2、创建构建目录
在项目根目录外创建一个构建目录，例如，可以在项目根目录下创建一个名为build的文件夹

mkdir build             # 创建构建目录
cd build                # 进入构建目录

3、运行CMake

在构建目录中运行CMake命令，指向项目根目录

cmake ..

4、编译项目
在构建目录中运行make命令

make
